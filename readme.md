# PHP routing
A basic PHP routing system I made back in 2019. The reason this project exists is that I needed a primitive, "no dependency", and readonly routing system for school. The project should be easy to understand for beginner PHP developers, you can fork the project and make it your own.

The project contains an .htaccess file, you might be using a different kind of web server, make sure all requests are routed through index.php. This project should be PHP v8.1 compatible.
